# Wallet-Desktop

- Clone Repository "git clone https://github.com/Leax-Foundation/Wallet-Desktop.git"

```  cd in wallet-desktop ```

``` npm install ```

```  npm start ``` 

## Building Aplication

### Linux

``` npm run package-linux ```
  
### Mac

``` npm run package-mac ```
  
``` npm run create-installer-mac ```
  
  
### Windows

``` npm run package-win ```
